## 화면 구성

```
┌──────────────────────────────────────────────────┐
│  Dup Web                                 ☀ 테마  │  ← 헤더
├─────────────────┬────────────────────────────────┤
│  스캔 경로      │  일반 | 이미지 | 영상 | 아카이브 │  ← 탭
│  옵션           │                                │
│  [스캔 시작]    │  결과 그룹 목록                 │
│                 │                                │
│  로그           │                                │
│  진행률 게이지  │  [ZIP 불러오기] [ZIP 저장]      │
└─────────────────┴────────────────────────────────┘
```

- **왼쪽 사이드바** — 스캔 경로 설정, 옵션, 로그, 진행률
- **오른쪽 콘텐츠** — 탭별 중복 그룹 목록, 필터/정렬, 액션 버튼

## 스캔하기

### 기본 스캔

1. **경로 추가** 버튼을 누르거나 폴더 아이콘(📁)을 클릭해 탐색창에서 폴더를 선택합니다.
2. 경로를 여러 개 추가하면 모두 합쳐서 스캔합니다.
3. **스캔 시작** 버튼을 클릭합니다.

### 스캔 중

- 사이드바 로그에 단계별 진행 상황이 표시됩니다.
- 하단 **진행률 게이지**에 현재 단계와 처리된 파일 수가 실시간으로 나타납니다.
- 중단하려면 **중단** 버튼을 클릭합니다.

### 스캔 단계

| 단계 | 설명 |
|------|------|
| 파일 목록 수집 | 경로 내 모든 파일을 수집합니다 |
| 해시 중복 탐지 | SHA-256 해시로 완전히 동일한 파일을 찾습니다 |
| 이미지 유사도 분석 | pHash로 비슷한 이미지를 찾습니다 |
| 영상 유사도 분석 | 프레임 해시로 비슷한 영상을 찾습니다 |
| 아카이브 중복 분석 | ZIP/7z 내부 파일이 겹치는 압축파일을 찾습니다 |

## 결과 보기

스캔 완료 후 탭별로 결과가 분류됩니다.

| 탭 | 내용 |
|----|------|
| **일반** | 해시가 동일한 완전 중복 파일 |
| **이미지** | 유사한 이미지 (exact / similar 구분) |
| **영상** | 유사한 영상 |
| **아카이브** | 내부 파일이 겹치는 압축 파일 |

### 그룹 정보

각 그룹 헤더에는 다음 정보가 표시됩니다:

- **파일 수** — 그룹 내 파일 개수
- **절약 가능 용량** — 중복 파일을 삭제했을 때 확보되는 용량
- **이미지/영상** — 완전동일 / 유사 구분
- **아카이브** — 공통 파일 수 · 내부 파일 수

### 원본 추정

각 그룹에서 **원본 추정** 배지가 붙은 파일이 자동으로 KEEP으로 설정됩니다.  
원본 추정 기준:
- 경로 깊이가 얕을수록 우선 (상위 폴더에 있는 파일)
- 파일명에 `copy`, `복사`, `(1)`, `_20231015` 같은 패턴이 없는 파일

> 자동 추정이 맞지 않으면 클릭해서 직접 변경할 수 있습니다.

## 삭제 표시하기 (KEEP / REMOVE)

삭제를 실행하기 전에 각 파일을 **KEEP**(유지) 또는 **REMOVE**(삭제 예정)로 표시합니다.  
실제 삭제는 별도 버튼으로 실행하며, 표시만 해서는 파일이 삭제되지 않습니다.

### 개별 파일 표시

- 파일 행을 **클릭**하면 KEEP → REMOVE → (미설정) 순으로 전환됩니다.
- **초록색** — KEEP (유지)
- **빨간색** — REMOVE (삭제 예정)

### 일괄 표시

필터 입력란에 경로/파일명을 입력한 뒤:

- **필터 내 일괄 KEEP** — 필터에 걸린 파일을 모두 KEEP으로 설정
- **필터 내 일괄 REMOVE** — 필터에 걸린 파일을 모두 REMOVE로 설정

## 필터와 정렬

### 필터

경로/파일명 필터 입력란에 텍스트를 입력하면 해당 문자열이 포함된 파일이 있는 그룹만 표시됩니다.

### 정렬

| 옵션 | 설명 |
|------|------|
| 정렬 없음 | 스캔된 순서 |
| 절약 용량↓ | 삭제 시 확보 용량이 큰 순 |
| 최대 파일 크기↓ | 그룹 내 가장 큰 파일 기준 |
| 파일 수↓ | 그룹 내 파일이 많은 순 |
| 아카이브 우선 | 압축 파일이 포함된 그룹 먼저 |
| 일치도↓ | 아카이브 탭 전용, 공통 파일이 많은 순 |

### 보기 필터 (일반 탭)

- **전체** — 일반 파일과 아카이브 내부 파일 모두 표시
- **일반 파일만** — 아카이브 내부 항목 제외
- **아카이브 내부만** — 압축 파일 안에서 발견된 중복만 표시

## 파일 비교 보기

이미지 / 영상 탭의 그룹에서 파일 행 오른쪽 **비교** 버튼을 클릭하면 나란히 비교 패널이 열립니다.

- **이미지** — 두 이미지를 나란히 표시
- **영상** — 두 영상을 동시 재생 (재생/일시정지 연동)

## 프리셋

자주 사용하는 스캔 경로와 옵션 조합을 저장해두고 불러올 수 있습니다.

1. **☰ 버튼** (스캔 시작 옆)을 클릭해 프리셋 모달을 엽니다.
2. **현재 설정 저장** — 현재 경로·옵션을 이름을 붙여 저장합니다.
3. 저장된 프리셋을 클릭하면 즉시 경로와 옵션이 불러와집니다.
4. **내보내기 / 가져오기** — 프리셋을 JSON 파일로 저장하거나 다른 기기에서 가져옵니다.

## 세션 저장과 불러오기 (ZIP)

스캔 결과와 KEEP/REMOVE 표시를 파일로 저장해 나중에 이어서 작업할 수 있습니다.

### 저장

1. **ZIP 저장** 버튼을 클릭합니다.
2. 저장 위치와 파일명을 선택합니다.
3. 스캔 결과가 `.zip` 파일로 저장됩니다.

### 불러오기

1. **ZIP 불러오기** 버튼을 클릭합니다.
2. 이전에 저장한 `.zip` 파일을 선택합니다.
3. 결과와 KEEP/REMOVE 표시가 복원됩니다.

> **주의** — ZIP 파일 내부의 CSV를 직접 수정하면 불러오기가 실패할 수 있습니다.

## 삭제 실행하기

> **주의** — 삭제된 파일은 휴지통으로 이동하지 않고 즉시 영구 삭제됩니다.

### 선택 삭제

현재 탭에서 필터에 걸린 그룹 중 REMOVE로 표시된 파일만 삭제합니다.

### 전체 REMOVE 삭제

현재 탭 전체에서 REMOVE로 표시된 모든 파일을 삭제합니다.

### 삭제 전 확인

- 삭제 버튼을 누르면 대상 파일 목록과 함께 확인 다이얼로그가 표시됩니다.
- **파일 존재 확인** 버튼으로 이미 없어진 파일을 목록에서 제거할 수 있습니다.

### 전체 초기화

스캔 결과와 KEEP/REMOVE 표시를 모두 지웁니다. 실제 파일은 삭제되지 않습니다.

## 고급 옵션

**고급 옵션** 펼침 메뉴에서 탐지 민감도를 조정할 수 있습니다.

### 이미지 옵션

| 옵션 | 기본값 | 설명 |
|------|--------|------|
| exact 임계값 | 0 | 이 값 이하의 pHash 거리를 완전동일로 판단. 높일수록 더 많이 탐지 |
| similar 임계값 | 10 | 이 값 이하를 유사로 판단. 높일수록 더 많이 탐지 |

### 영상 옵션

| 옵션 | 기본값 | 설명 |
|------|--------|------|
| exact 임계값 | 0 | 완전동일 판단 기준 |
| similar 임계값 | 10 | 유사 판단 기준 |
| 샘플 프레임 수 | 10 | 비교에 사용할 프레임 수. 많을수록 정확하나 느림 |

### 아카이브 옵션

| 옵션 | 기본값 | 설명 |
|------|--------|------|
| 최소 겹침 파일 수 | 5 | 공통 파일이 이 수 이상인 경우만 그룹으로 표시 |
| 최소 내부 파일 수 | 0 | 아카이브 내부 파일이 이 수 이상인 경우만 분석 (0 = 제한 없음) |

### 제외 패턴

스캔에서 제외할 파일 패턴을 줄바꿈으로 구분해 입력합니다. `fnmatch` 형식을 사용합니다.

```
*.tmp
.DS_Store
**/Thumbs.db
**/desktop.ini
```

### 스캔 제외 옵션

- **이미지 skip** — 이미지 유사도 분석을 건너뜁니다
- **영상 skip** — 영상 유사도 분석을 건너뜁니다 (시간이 오래 걸릴 때 유용)
- **압축 skip** — 아카이브 내부 분석을 건너뜁니다
